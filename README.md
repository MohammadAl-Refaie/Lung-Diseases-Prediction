# Lung-Diseases-Prediction

This project uses a pre-trained MobileNet model to classify lung diseases from chest X-ray images.
It distinguishes between three classes:

1- Normal

2- Lung Opacity

3- Viral Pneumonia

A user-friendly GUI built with Tkinter allows users to upload chest X-ray images and instantly receive predictions, confidence scores, and the displayed image.
